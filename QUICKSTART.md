# Быстрый старт

Краткое руководство по запуску приложения за 5 минут.

## Шаг 1: Установка зависимостей

### Ubuntu/Debian
```bash
sudo apt-get install build-essential cmake qt6-base-dev
```

### Fedora
```bash
sudo dnf install gcc-c++ cmake qt6-qtbase-devel
```

### macOS
```bash
brew install cmake qt@6
```

## Шаг 2: Сборка

```bash
mkdir build
cd build
cmake ..
cmake --build .
```

## Шаг 3: Запуск

```bash
./BaggageSystem
```

## Первое использование

1. **Создайте новый файл**: Нажмите "1. Создать файл"
2. **Добавьте записи**: Нажмите "6. Добавить запись"
   - № рейса: `SU1234`
   - Ф.И.О.: `Иванов Иван Иванович`
   - Количество вещей: `2`
   - Веса: `20.5` и `25.0`
3. **Сохраните**: Меню "Файл" → "Сохранить"

## Нужна помощь?

Если возникли проблемы при сборке:
1. Проверьте версию Qt: `qmake --version` (требуется 5.15+)
2. Проверьте версию CMake: `cmake --version` (требуется 3.16+)
3. Проверьте компилятор: `g++ --version` (требуется поддержка C++17)
